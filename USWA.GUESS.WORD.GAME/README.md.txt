# Word Guessing Game

A Python game where players need to guess a word by selecting letters. It gives feedback on correct or incorrect guesses.

## Features
- Random word selection from a predefined list
- Player feedback on correct/incorrect guesses
- Keeps track of attempts

## How to Play
1. Run the `word_guess.py` script in your Python environment.
2. You’ll be asked to guess letters to complete the word.
3. You’ll receive feedback after each guess.

## Requirements
- Python 3.x
