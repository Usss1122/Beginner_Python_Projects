# Hangman Game

A simple Python implementation of the classic Hangman game where players guess letters to complete a word.

## Features
- Random word selection
- Player input to guess letters
- Display of remaining attempts

## How to Play
1. Run the `hangman.py` script in your Python environment.
2. The game will prompt you to guess letters.
3. Keep guessing until you either complete the word or run out of attempts.

## Requirements
- Python 3.x
